# Loantractor

**Automated Loan Data Extraction SaaS**

Loantractor is a secure, AI-powered tool that extracts structured data from borrower documents (Pay Stubs, Tax Returns, Loan Applications) with high accuracy.

## Features
- **AI Extraction**: Uses OpenAI GPT-4o Vision to parse documents.
- **Context-Aware**: Optimizes extraction based on document type.
- **Secure**: SOC2 compliant infrastructure via Supabase.

## Tech Stack
- Frontend: React + Vite + TypeScript
- Styling: TailwindCSS
- Backend: Supabase (Auth, Database, Storage)
- Deployment: Netlify
